# Getting Started

This page will explain how to start playing CozyVille.

## What is CozyVille?

CozyVille is an online crypto game with farming, trading, crafting and earning systems.

## First Steps

1. Create your account
2. Learn the basic controls
3. Complete the starter tasks
4. Start earning gold
5. Explore the marketplace
